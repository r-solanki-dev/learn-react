1. What is "conditional rendering"?
   Using in-component logic to determine what elements to show on the page.


2. When would you use &&?
   When the condition evaluates to a boolean true or false, and, when there is only an "if" condition.



3. When would you use a ternary?
   When the logic requires "if, else"



4. What if you need to decide between > 2 options on
   what to display?
   Place the conditional logic outside the main return statement.
